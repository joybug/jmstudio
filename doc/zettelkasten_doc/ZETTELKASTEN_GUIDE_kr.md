# 🕸️ 제텔카스텐(Zettelkasten) 및 지식 그래프 뷰 사용 가이드

**Joy Markdown Studio (JM-STUDIO)** v3.7.6부터 도입된 핵심 기능인 **쌍방향 위키 링크**와 **지식 그래프 뷰(Graph View)**를 사용하여 당신의 노트와 논문들을 유기적으로 연결하는 방법을 안내합니다.

---

## 1. 쌍방향 링크 (위키 링크) 생성하기

마크다운 에디터 화면에서 다른 문서의 이름(확장자 `.md` 제외)을 대괄호 두 개 `[[ ]]` 로 감싸기만 하면 자동으로 해당 문서로 연결되는 하이퍼링크가 생성됩니다.

### 기본 문법 (단순 링크)
```markdown
오늘 연구에서는 [[양자역학]]의 기본 원리를 응용했다.
```
> **결과:** 미리보기 화면에서 `양자역학`이라는 보라색 글씨가 나타나며, 클릭하면 즉시 `양자역학.md` 파일이 열립니다. 만약 파일이 없다면 새로 만들지 물어보고 빈 문서를 즉시 만들어 줍니다.

### 별칭 문법 (표시될 이름 바꾸기)
실제 파일 이름과 화면에 보여줄 단어를 다르게 하고 싶을 때는 `|` (파이프) 기호를 사용합니다.
* `[[파일이름|화면에 보여줄 글자]]`

```markdown
이 반응은 [[Chemical_Reaction_01|친핵성 치환 반응]]을 따릅니다.
```
> **결과:** 클릭하면 `Chemical_Reaction_01.md` 파일이 열리지만, 화면에는 `친핵성 치환 반응`이라고 예쁘게 표시됩니다.

## 2. 🕸️ 지식 그래프 뷰 (Graph View) 활용하기

내가 작성한 문서들이 서로 어떻게 연결되어 있는지 한눈에 볼 수 있는 마법 같은 기능입니다.

### 실행 방법
1. JM-STUDIO 상단 툴바(미리보기 버튼 오른쪽)에 있는 **보라색 🕸️ (Graph)** 버튼을 클릭합니다.
2. 에디터와 미리보기 화면 위로 우주 공간처럼 아름다운 **3D 물리 엔진 그래프 화면**이 부드럽게 펼쳐집니다.

### 조작 방법
* **마우스 휠 스크롤**: 화면을 확대(Zoom In)하거나 축소(Zoom Out)할 수 있습니다.
* **배경 드래그**: 캔버스를 잡고 드래그하여 화면 전체를 이동시킬 수 있습니다.
* **노드(공) 드래그**: 특정 문서의 노드를 마우스로 잡아 끌면 탄성력 있게 움직이며, 원하는 곳에 던져놓을 수 있습니다.
* **노드(공) 클릭**: 특정 문서 공을 마우스 왼쪽 버튼으로 클릭하면, 그래프가 즉시 닫히면서 **해당 마크다운 문서가 에디터에 열립니다.**

### 🎨 노드 색상과 크기 분류 규칙

지식 그래프는 노드의 **색상**과 **크기**를 통해 문서의 성격과 연결 중요도를 직관적으로 시각화합니다.

* 🟡 **황금색 (Gold):** 여러 문서에서 **5개 이상 링크**가 연결된 최고 중요 핵심 문서 (지식의 중심 허브)
* 🔵 **하늘색 (Sky Blue):** `doc/` 폴더 내에 저장된 다양한 **가이드 및 참고 문서**
* 🟢 **초록색 (Emerald):** `docs/` 폴더 내에 저장된 **상세 참고 자료**
* 🟣 **보라색 (Purple):** 워크스페이스 최상위(루트) 폴더에 있는 **일반 노트 및 문서**
* 🔴 **빨간색 (Red):** 링크로 언급은 되었으나 **아직 실제 파일이 생성되지 않은 미완성 문서** (클릭 시 즉시 새 파일 생성 제안)

> **노드 크기의 의미:** 문서가 다른 문서들과 **연결(참조)된 횟수가 많을수록** 공의 크기가 실시간으로 팽창합니다. 황금색으로 빛나는 가장 큰 공들은 당신의 노트 데이터베이스에서 가장 핵심이 되는 중심 주제어입니다.
>
> **지식의 흐름 애니메이션:** 지식이 유기적으로 순환하는 구조를 표현하기 위해 연결선을 따라 **핑크색 네온 광원 파티클이 실시간으로 흘러가며** 지식의 참조 방향을 한눈에 보여줍니다.

## 💡 활용 예제 (이렇게 따라 해보세요!)

당신의 작업 폴더에 아래와 같이 3개의 마크다운 파일을 만들고 내용을 복사해 보세요.

### 📝 파일 1: `상대성이론.md`
```markdown
# 특수 상대성 이론
아인슈타인이 제안한 이 이론은 시간과 공간이 절대적이지 않음을 설명합니다.
더 깊은 내용은 [[로런츠_변환]]을 참고하세요. 그리고 이는 [[광전효과]] 연구와도 같은 시기에 진행되었습니다.
```

### 📝 파일 2: `로런츠_변환.md`
```markdown
# 로런츠 변환
관성계 사이의 좌표 변환식입니다. 이 식을 통해 [[상대성이론|특수 상대성 이론]]의 시간 지연 현상을 유도할 수 있습니다.
```

### 📝 파일 3: `광전효과.md`
```markdown
# 광전효과
빛을 쪼였을 때 금속 표면에서 전자가 튀어나오는 현상입니다. 이 현상은 고전 전자기학으로 설명할 수 없었으며, [[상대성이론]]의 창시자인 아인슈타인이 광양자 가설로 설명해냈습니다.
```

### 🧪 결과 확인
위 세 문서를 만들고 나서 상단의 **🕸️ 그래프 버튼**을 눌러보세요!
`상대성이론`, `로런츠_변환`, `광전효과` 세 개의 공(Node)이 서로 거미줄처럼 화살표로 연결되어 아름답게 상호작용하는 모습을 즉시 확인할 수 있습니다. 공을 클릭하며 문서를 넘나들며 연구에 몰입해 보세요!
